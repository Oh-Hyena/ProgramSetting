[ anaconda 가상환경 만들기 ]
https://89douner.tistory.com/73?category=878197

anaconda prompt 실행
->conda create -n test python=3.9 (가상환경 만들기)
->conda activate test (가상환경 활성화하기)
->conda list (패키지 확인하기)
->pip install numpy (패키지 설치하기)
->conda env remove -n test (가상환경 삭제하기)

anaconda navigator 실행
->environment ->name : test / python = 3.9 ->creat (가상환경 만들기)
->not installed ->numpy -> apply (패키지 확인하고 설치하기)
